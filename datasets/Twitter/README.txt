Twitter "buzz prediction" task (ridge regression)
-------------------------------------------------

A. Obtain the dataset:

1. Download twitter dataset from:
   http://ama.liglab.fr/resourcestools/datasets/buzz-prediction-in-social-media/
   The regression task download is a zip file called regression containing two folders, Twitter and TomsHardware.

2. Copy the file Twitter/Twitter.data into this folder.

3. Run python3 process_twitter.py
   This produces a dataset with re-normalized and shifted data values called twitter_dataset.txt.

--------------------------------

B. Run the ridge regression code on this dataset file.
